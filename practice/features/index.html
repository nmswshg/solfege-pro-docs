指板トレーニング（指板トレーニング）

ギター/ベースの指板上で、指定された音名の位置を素早く特定するトレーニング。

  • 範囲設定：弦・フレット範囲を細かく設定可能
  • 応答時間測定：正解までの時間を記録
  • 弱点分析：苦手なポジションを統計で表示
  • プリセット保存：練習設定を保存して再利用
  • マルチチューニング対応：標準、Drop D、Open G、変則チューニングまで切替可能
  • 弦の絞り込み：苦手な弦だけを集中練習
  • ノートモード：「音名 → 位置」「位置 → 音名」の双方向

→ ガイドで深掘り：オクターブシェイプとフレット範囲ごとの攻略

インターバル認識（音程トレーニング）

2つの音を聴いて、その音程（インターバル）を当てるトレーニング。

  • 音程選択：練習したい音程だけを選択可能
  • 上昇/下降：上昇音程、下降音程、両方から選択
  • ランダム基準音：毎回違うキーで出題
  • ピアノ音源：高品質なピアノサウンド
  • 13 種類のインターバル：1 度から完全 8 度までを段階的にカバー
  • 4 つの再生モード：上昇 / 下降 / 同時 / ハーモニックを切替
  • 苦手音程の重点出題：間違えやすい音程の出題頻度を自動で上げる

→ ガイドで深掘り：相対音感の作り方と 13 インターバルの優先順位

コード認識（コード認識）

コードを聴いて、そのコードタイプを当てるトレーニング。

  • 3和音：Major, minor, dim, aug
  • 4和音：7th, maj7, m7, m7(♭5)
  • その他：sus4, add9 など
  • 転回形：ルートポジション以外の転回形にも対応
  • 20 種類以上のコードタイプ：トライアド・7th・テンション・sus 系まで網羅
  • 5 段階の難易度設定：Major/minor の 2 択から、20+ 種類全部までを段階的に
  • ブロック / アルペジオ再生：分析にはアルペジオ、スピード向上にはブロック
  • 比較再生：候補コードを並べて聴き比べられる
  • 苦手克服モード：間違えやすいコードに出題を寄せる

→ ガイドで深掘り：3 度の聴き方と段階的なコード認識の積み上げ

スケール認識（スケール認識）

スケールを聴いて、そのスケール名を当てるトレーニング。チャーチモードと特性音の聴き取りに対応。

  • Major / minor：基礎の 2 種から開始
  • チャーチモード：Ionian / Dorian / Phrygian / Lydian / Mixolydian / Aeolian / Locrian の 7 モードを段階的に追加
  • 特性音にフォーカス：モードを決定づける 1 音を意識して聴く
  • 段階的難易度：選択肢を 2 → 3 → 7 と増やしていける

→ ガイドで深掘り：モード特性音の聴き方と段階的な学習順序

コード進行トレーニング（コード進行トレーニング）

コード進行を聴いて、各コードの機能 (T/SD/D)・度数・コード名のいずれかを答えるトレーニング。

  • 15 種類以上の進行パターン：I-V-vi-IV, ii-V-I, I-IV-V-I などポップス／ジャズの定番
  • 3 つの回答形式：Function (T/SD/D)、ローマ数字 (I/IV/V…)、Chord Symbol (C/F/G…)
  • 2〜4 コードの進行長：短い進行で基礎、長い進行で実戦感
  • トニックプレビュー：出題前に I を聴いてキー感覚を設定 (ON/OFF 切替)
  • Pop / Jazz カテゴリ：ジャンルで出題を絞り込める
  • パターン別統計：進行ごとの正答率を可視化

→ コード進行ガイド：機能和声の概念と段階的な聴き方

→ コード機能トレーニング：解決感から 4 小節進行までの 4 段階実践プラン

リズムトレーニング（リズムトレーニング）

クリック音に合わせてタップ/クラップし、リズム精度を測定。

  • タップモード：画面タップで測定（静かな環境向け）
  • クラップモード：マイクで手拍子を検出（体全体のグルーヴ向け）
  • BPM設定：40〜200BPMまで対応
  • 統計表示：平均誤差、走り/モタり傾向を数値化
  • キャリブレーション：デバイスの遅延を自動補正
⚠️ Bluetoothイヤホン非推奨
AirPodsなどのBluetoothイヤホンはコーデックや機種により数十〜100 ms 以上の遅延が出ることがあります。正確な測定のため、有線イヤホンまたは内蔵スピーカーをご使用ください。

→ リズム感を深く理解する：前乗り・後乗り・ポケットのガイド

譜読みトレーニング（譜読みトレーニング）

五線譜上の音符を見て、音名を素早く答えるトレーニング。

  • ト音記号/ヘ音記号：両方に対応
  • 範囲設定：オクターブ範囲を設定可能
  • 臨時記号：シャープ・フラットの有無を選択
  • 応答時間測定：読譜速度を記録
  • 全 12 キー対応：キー別に出題範囲を絞れる
  • 加線の ON / OFF：加線エリアの出題を切替
  • 苦手克服モード / エラー分析：間違いやすい音を集中出題、エラー傾向を可視化

→ ガイドで深掘り：ランドマーク 5 音法と段階的な拡張
